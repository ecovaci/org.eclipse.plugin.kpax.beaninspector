Select the JavaBean

The first thing to do is to select a Java type from JavaBean Inspector dialog.
There are two ways to do that:

  1. Open the JavaBean Selector dialog directly, using the shortcut key, default Ctrl+Alt+8)
  2. Use the toolbar button (or the associated shortcut key, default Ctrl+Alt+0)
