About

This plugin can be used for JavaBean inspection, selecting paths within the selected JavaBean and path validation. It is especially useful when JavaBean objects are binded to external structures, like Javascript objects. See:

  • JavaBean Expression Paths
  • Select JavaBean
  • Contextual Path Selector
  • Path Validation
